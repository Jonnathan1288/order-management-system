# Order Management System

Description

- Sample project for managing orders, including database schemas and UML diagrams.

Requirements

- Docker or a Postgres server (see the scripts in the `database` folder).
- Git

Installation

1. Clone the repository:

    git clone [<repo-url>](https://github.com/Jonnathan1288/order-management-system.git)

2. Review the scripts in `database/schemas` to create the database.

Usage

- Documentation and diagrams are available in the `docs/` folder.
- Run the SQL scripts in `database/schemas` to create the tables.

Project Structure

- `database/` — SQL scripts and schemas
- `docs/` — documentation and UML diagrams
